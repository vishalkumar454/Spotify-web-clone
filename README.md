# Spotify Clone - Web Player UI

A simple and clean Spotify Web Player interface clone built using **HTML**, **CSS**, and **Font Awesome icons**.  
This project is focused on front-end UI/UX design and does not include music playback functionality.

---

## Features

- **Navigation Bar** with Search, Home & Install App Option  
- **Sidebar** including Library and Playlist options  
- **Main Content Area** showing music cards (Recently Played, Trending, Charts)  
- **Music Player Bar** at the bottom with control icons  
- Smooth **hover animations** and **tooltips**  
- Fully **responsive layout** adjustments using Flexbox
